# Portfólio de Social Media

Este é um site simples de portfólio para exibir trabalhos em social media, criado com HTML, CSS e JavaScript.

## Como usar

1. Abra o arquivo `index.html` em um navegador web para visualizar o site.
2. Para editar o conteúdo:
   - Modifique o texto em `index.html`.
   - Adicione suas imagens na pasta e atualize os `src` das tags `<img>`.
   - Ajuste os estilos em `style.css` se necessário.
   - Adicione mais funcionalidades em `script.js`.

## Estrutura

- `index.html`: Estrutura principal do site.
- `style.css`: Estilos com cores dourado e marrom médio.
- `script.js`: JavaScript para interatividade básica (toggle de descrições).

## Personalização

- Cores: Dourado (#FFD700) e marrom médio (#A0522D).
- Adicione mais seções `.trabalho` no HTML para mais projetos.
- Atualize as informações de contato na seção correspondente.

Para abrir no VS Code, navegue até a pasta `portfolio-social-media` e abra-a como workspace.
